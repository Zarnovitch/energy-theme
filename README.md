# Energy Theme

A dark, Material-inspired theme designed for comfortable coding in low-light environments.

## Features

- Dark mode with a refined Material-inspired palette
- Optimized for web development (React, Vue, Nuxt.js, Tailwind, Next.js)
- Accessibility-conscious colors, including colorblind-friendly contrasts
- Compatible with VS Code, VSCodium, and other VS Code-based editors

## Installation

### Via VSIX

1. Download the `.vsix` file from the release or build using `vsce package`.
2. Open your editor → **Extensions** (`Ctrl+Shift+X`) → **Install from VSIX…**
3. Select the downloaded `.vsix` file and install.

### Via Marketplace

Search for **Energy Theme** in the Extensions panel and click **Install**.

## Usage

1. Open the Command Palette (`Ctrl+Shift+P`).  
2. Select **Preferences: Color Theme**.  
3. Choose **Energy Theme** from the list.
